MANUAL DE CSS

Que es css

El código CSS hace la vida más fácil al desarrollador front-end al separar las estructura de un documento HTML de su presentación. Dicho de otro modo el HTML actuaría como es esqueleto de la web, definiendo su estructura básica, y el CSS añadiría toda la capa de personalización sobre el que la web define su aspecto final. Siguiendo este fundamento resulta muy fácil para un diseñador web realizar cambios en la apariencia de una web sin afectar de manera dramática a su contenido. El contenido siempre será el mismo, solo cambia como aquello que podemos ver. CSS es fácil de entender y aprender, y nos da un potente control de cómo diseñar los documentos HTML.

COLORES CSS

Elemento Descripción Sintaxis
color Podemos especificar un color usando un nombre predefinido. Por ejemplo podemos especificar que el color del párrafo sea rojo con la siguiente regla p{ color: red; }
rgb Para definir un color como la mezcla las intensidades de los colores primarios podemos usar la función elemento{ color: rgb(20,20,20); }
rgba Podemos establecer el valor de un color mezclando diferentes intensidades de los tres colores primarios. elemento{ color: rgba(20,20,20,1); }
hexadecimal Podemos utilizar un número hexadecimal para definir un color. Es una alternativa a la función rgb, es decir permite definir un color como la mezcla de intensidades de los colores primarios elemento{ color: #numero_hexadecimal; }
hsl En este modelo utilizamos conceptos que son más fáciles de comprender. En CSS para poder trabajar con este modelo tenemos la siguiente opción: elemento{ color: hsl(h,s,l); }
hsla Podemos definir la transparencia de un color usando la función hsla, esta función es idéntica a la función hsl, pero con un parámetro adicional denominado opacidad, para especificar la transparencia. elemento{ color: hsla(h,s,l,a); }

SELECTORES

Elemento Descripción Sintaxis
Selector Universal ( * ) Selecciona todos los elementos *{ declaraciones; }
Selector de Tipo Permite seleccionar elementos usando el nombre del elemento elemento{ declaraciones; }
Selector de clase Permite seleccionar elementos en función de los valores del atributo class de los elementos. Para que los elementos sean seleccionados, su atributo class debe incluir un valor que coincida con el valor del selector p class=”foo bar”> ¡HOLA MUNDO!

p class=”foo baz” > ¡ADIÓS MUNDO!

.foo{ declaraciones; }
Selector de ID Permite seleccionar un elemento en función del valor del atributo ID del elemento. Para que el elemento sea seleccionado, su atributo id debe coincidir exactamente con el valor dado en el selector. #foo{ declaraciones; }

Descripción Sintaxis
Selecciona elementos que tengan un atributo con el nombre attr selector[attr] { … }
Selecciona elementos que tengan un atributo con el nombre attr. El único valor del atributo es exactamente value selector[attr=value] { … }
Selecciona los elementos que tengan un atributo con el nombre attr. Uno de los valores del atributo es exactamente value selector[attr~=value] { … }
Selecciona los elementos que tengan un atributo con el nombre attr. El primero de los valores del atributo empieza con value selector[attr ^= value] { … }
Selecciona los elementos que tengan un atributo con el nombre attr. El último de los valores del atributo termina con value selector[attr $= value] { … }
Selecciona los elementos que tengan un atributo con el nombre attr. Uno de los valores del atributo contiene al menos una ocurrencia de value selector[attr *= value] { … }

Elemento Descripción Sintaxis
Pseudo-Classes Permite seleccionar elementos en base a su estado (:disabled), posicion de los elementos en el documento(:nth-child) e interacción del usuario(:hover) selector:pseudo-class { … } button:hover{color: blue;}
Pseudo-Elementos Seleccionan parte de un elemento en lugar del elemento mismo. Los pseudo-elementos se añaden a los selectores y permiten añadir estilos a una parte concreta de los elementos seleccionados selector::pseudo-elemento { … } p::first-line{color: blue;}
Unión de Selectores Selecciona la unión de los elementos seleccionados por cada selector de la lista Selector-1, Selector-2, … { }
Seleccionar Elementos Descendientes Seleccionamos todos los elementos seleccionados por B que son descendientes de los elementos seleccionados por A A B {}
Seleccionar Elementos Hijos Seleccionamos todos los elementos seleccionados por B que son hijos de los elementos seleccionados por A A>B{}
Seleccionar Elementos Hermanos Seleccionamos todos los elementos seleccionados por B que son hermanos de los elementos seleccionados por A A~B {}
Seleccionar Elementos Hermanos Adyacentes Seleccionamos todos los elementos seleccionados por B que son hermanos y que están inmediatamente despues de los elementos seleccionados por A A+B {}
